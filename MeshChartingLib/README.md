MeshChartingLib provides raylib-free, GL-free utilities for mesh chart segmentation and UV atlas packing: `build_adjacency` (edge-welded triangle adjacency), `segment_charts` (region-growing normal-cone partitioning), `plane_basis` (robust orthonormal tangent/bitangent from a normal), and `pack_charts` (shelf-packing chart rects into a texel atlas). These were salvaged from the chart-based imposter cage in MatterSurfaceLib and placed here for independent reuse. Run tests with `make -C tests run` from this directory.
